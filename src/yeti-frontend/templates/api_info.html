CryptoCurrency API

Overview

The Cryptocurrency API is a freely available RESTful API built using the CoinGecko API. Written in python, this API can be used to get the recent performance of certain cryptocurrencies.
The information for each coin available with the API:
  1. Current Price
  2. One Minute Percentage
  3. Ten Minute Percentage
  4. Thirty Minute Percentage
  5. One Hour Percentage
  6. Six Hour Percentage
  7. Twelve Hour Percentage
  8. One Day Percentage
  9. One Week Percentage
  10. Market Capital
  11. 24 Hour Volume
  12. Timestamp

Example usage

There are 2 ways of using the API.
  1. Retreive information for specific Coins
  2. Retrieve information for all coins
Retreiving information for specific coins
Performing the following HTTP request will return a JSON object with the information for the coin specified:

http://hostname/api/coin_data/history?name=bitcoin

Retreiving information for all coins
Performing the following HTTP request will return a list of JSON objects for each of the coins:

http://hostname/api/coin_data/history/all
